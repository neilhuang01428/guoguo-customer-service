iPad 巧控鍵盤上手指南

你剛拿到 iPad 的巧控鍵盤，過去只用過 Windows、沒碰過 Apple 筆電。別擔心——這份用 80/20 法則，只教最關鍵的 20%，讓你從小白快速變熟手。

整份只圍繞一個心法展開：把你熟悉的 Ctrl 換成 ⌘，八成快捷鍵你其實早就會了。剩下的是幾個 Apple 專屬的貼心習慣。花 10 分鐘讀完，今天就能上手。

core_idea

00 · 一個核心心法

Windows 快捷鍵的靈魂是 Ctrl：複製 Ctrl+C、貼上 Ctrl+V…。到了 iPad，這個角色換成 ⌘（Command，指令鍵）。其他都一樣，只是把 Ctrl 改按 ⌘。

“

把 Ctrl 換成 ⌘，你已經會了八成。

剩下兩成，是 Apple 才有的貼心習慣——這份幫你補齊。
⌘ 在哪
就在空白鍵的左右兩側——差不多是你在 Windows 習慣按 Alt 的位置。左右各一顆，用哪顆都行。
神技 · 不用背
在任何 App 裡長按 ⌘ 不放，畫面會跳出「這個 App 的所有快捷鍵小抄」。忘了就長按它，比查 Google 快。

01 · 修飾鍵對照（Windows → iPad）

「修飾鍵」就是要搭配別的鍵一起按的那幾顆。先把這張對照記起來，後面全部通：

你想做的Windows 你按iPad 巧控鍵盤按
大部分快捷鍵Ctrl⌘ Command（主角）
次要選項 / 特殊符號Alt⌥ Option
大寫 / 加選Shift⇧ Shift（一樣）
少數系統操作Ctrl⌃ Control（少用）
切換中英 / 表情符號—🌐 地球鍵
最容易搞混的一點
Windows 的 Ctrl 在 iPad 不是那顆 ⌃ Control，而是 ⌘。iPad 上的 ⌃ Control 很少用，別被名字騙了。
80/20 快捷鍵

02 · 必背快捷鍵（這排就夠九成場景）

下面每一個，都和 Windows 一模一樣，只是 Ctrl 改按 ⌘。先把這排練熟，你就脫離小白了。

動作iPad 按鍵Windows 對照
複製⌘+CCtrl+C
貼上⌘+VCtrl+V
剪下⌘+XCtrl+X
復原（做錯了救回來）⌘+ZCtrl+Z
重做⌘+⇧+ZCtrl+Y
全選⌘+ACtrl+A
存檔⌘+SCtrl+S
在頁面內找字⌘+FCtrl+F

這三個是 iPad 專屬、但超好用

動作iPad 按鍵類似 Windows 的…
切換 App（跳到上一個）⌘+TabAlt+Tab

03 · 打字與游標移動（沒有 Home / End 的解法）

Windows 用戶最容易卡在這：巧控鍵盤沒有 Home、End、PageUp/Down，也沒有向前刪除鍵。別找了，Apple 用組合鍵取代，其實更順：

你想做的iPad 按鍵（Windows 舊習慣）
游標一次跳一個字（詞）⌥+← / →Ctrl+←→
跳到行首 / 行尾⌘+← / →Home / End
跳到文件最上 / 最下⌘+↑ / ↓Ctrl+Home/End
一次刪一整個詞⌥+⌫Ctrl+Backspace
選取（配任何上面的移動）加按 ⇧Shift+移動
規律，不是死背
移動的邏輯：⌥＝跳一個詞、⌘＝跳到底（行首尾／文件頭尾）。想「選起來」就多壓一顆 ⇧。三顆排列組合，全部搞定。
Apple 的貼心習慣
trackpad

04 · 觸控板手勢（巧控鍵盤的隱藏武器）

巧控鍵盤下方有一塊觸控板，不用一直伸手戳螢幕。手勢和 Mac 筆電一樣，記這幾個就順：

1指

移動 · 點按

單指滑動移動游標；按一下＝點擊。游標是個小圓點。

2指

捲動 · 右鍵 · 縮放

兩指上下滑＝捲動頁面；兩指點一下＝右鍵選單；兩指開合＝放大縮小。

3指

切 App · 回桌面

三指往上滑＝回主畫面；三指左右滑＝切換前後開的 App。

你會愛上的小細節
游標移到按鈕或文字附近時，會自動「吸」上去變形——不用像滑鼠那樣精準對準。這是 iPad 游標的貼心設計。
apple_habits

05 · 蘋果用戶都在用的好功能

這些不是快捷鍵，是「習慣」——學會了，你用起來就像老手：

萬用入口 · ⌘Space

Spotlight 搜尋

按 ⌘+Space，打幾個字就能開 App、找檔案、直接算數學（打 12*8）、查匯率、查單位。老手很少回桌面找 icon。

動口不動手

語音聽寫

按 🌐 地球鍵（或鍵盤上的麥克風），直接說話就變文字。打長訊息超快。

自動幫你

自動修正 · 預測字

打錯會自動修、還會猜你下個字。順著它按，速度更快。

⌘⇧3

截圖即標註

按 ⌘+⇧+3 截圖，左下角縮圖點一下就能直接畫重點、傳給客服。

一個習慣抵十個快捷鍵
養成「想做什麼就先按 ⌘+Space」的反射——找 App、找設定、算東西都從這裡開始。這是 Apple 老手和新手最大的差別。

06 · Windows 老習慣的坑

這幾個是 Windows 肌肉記憶會踩到的雷，先知道就不會慌：

你可能會找…其實 iPad 這樣
Ctrl+Alt+Del 關當掉的 App三指往上滑叫出 App 卡片 → 把那張往上撥掉
Alt+Tab 切視窗改成 ⌘+Tab（Alt 換成 ⌘）
Delete 想往前刪字⌫ 是往左刪；沒有內建往前刪，用方向鍵移過去再刪
找 Esc 鍵（有些款式沒有）按 ⌘+.（句號）當 Esc
找關機 / 鎖定鍵鍵盤上沒有；用 iPad 機身頂端的電源鍵
進階：切換中英文輸入法怎麼按？

按左下角 🌐 地球鍵循環切換；若你的款式沒有地球鍵，用 ⌃ Control+Space。長按地球鍵可挑特定輸入法。

PRO
